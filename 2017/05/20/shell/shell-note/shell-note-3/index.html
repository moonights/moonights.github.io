Shell传递参数

在执行shell脚本时，可以向脚本传递参数，脚本内获取参数的格式：$n，n代表数字编号，为脚本内需要获取的参数的编号。

$*与$@的异同：

都是应用所有参数

不同：只有在双引号中体现。假设在脚本运行时写了三个参数 1、2、3，，则 “ * “ 等价于 “1 2 3”（传递了一个参数），而 “@” 等价于 “1” “2” “3”（传递了三个参数）。

#!/bin/bash
echo "-- \$* demo ---"
for i in "$*";do
    echo $i
done
echo "-- \$@ demo ---"
for i in "$@"; do
    echo $i
done

执行效果：

$ chmod +x test.sh 
$ ./test.sh 1 2 3
-- $* 演示 ---
1 2 3
-- $@ 演示 ---
1
2
3
